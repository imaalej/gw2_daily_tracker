# GW2 Daily Tracker

An addon to track dailies that aren't tracked by the achievements menu such as world bosses, ley lines, hero choice chests, etc.

## Features

- World bosss completion status and next spawn countdown in UTC schedule
- Hero's Choice chest completion tracking
- Configurable refresh interval and keybind support
